  4. C++ Language

    A walk-through of the syntax of C++ using Pong code:

    • comments // and /* */ commenting practice
    • formatting and punctuation
    • data definitions const and variable
    • expressions: shift operators << and >> and %
    • variable scope: global and local
    • types: int, bool, unsigned char, and pointer types
    • array types and 2D arrays
    • literal values: decimal, hexadecimal, ASCII
    • subroutine declaration and definition
    • caller arguments
    • function parameters
    • if then else structures
    • ternary (condition)?a:b structure
    • loop structure: break and continue
    • switch case structure
    • array access and conversion to pointers

  5. Javascript Language

    A walk-through of the syntax of Javascript using Pong code:

    • comments as in C++
    • formatting and punctuation
    • data definition variable only private and public
    • same operators
    • variable scope: global and local
    • types: int, enumerations, platform specific e.g. Texture2D
    • compound types: Rect, Vector2, etc.
    • array types and initialization
    • literal values: decimal, hexadecimal, and enumerated values
    • subroutine declaration and definition
    • caller arguments
    • function parameters
    • if then else structures (same as C++)
    • loop structure: break and continue (same as C++)
    • switch case structure
    • array access, no pointers
    • global objects Input and GUI
