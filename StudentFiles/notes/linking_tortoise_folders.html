Linking Tortoise SVN folders

Here are set up instructions for your Tortoise SVN folders for the rest of the course. Using this layout will ensure that you don't accidentally commit changes to the wrong folder.

Programming folder

  • Make a new folder called "programming" in your "My Documents" folder.
  • Inside this, create the following new folders:
    • notes
    • Surname

Notes folder

  • Right click on the new "notes" folder and choose "SVN Checkout..."
  • In the "URL of repository" box enter:
    • https://lsbu-ame-5-cpg-2011-2012.googlecode.com/svn/trunk/2011-2012/notes

Your exercise folder

  • Right click on your surname folder and choose "SVN Checkout..."
  • In the "URL of repository" box enter:
    • https://lsbu-ame-5-cpg-2011-2012.googlecode.com/svn/trunk/2011-2012/your_surname
    • note: using https here means you can commit to this folder
